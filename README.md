# Spotify-Clone
A Spotify-inspired music player built with HTML, CSS, and Vanilla JavaScript featuring play/pause controls, next/previous navigation, progress bar seeking, synchronized song controls, and dynamic playlist management.

## Features
- 🎵 Play and pause songs
- ⏭️ Next and previous track navigation
- 🎚️ Interactive progress bar with seeking
- 🔄 Synchronized play/pause icons
- 🎬 Animated playing indicator
- 📱 Responsive layout

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## Future Improvements
- Volume control
- Shuffle and repeat modes
- Automatic playback of the next song
- Mobile optimization
